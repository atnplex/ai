# Perplexity Multi-Agent Coordination System

This folder contains coordination protocols and documentation for a 3-agent Perplexity Pro system using GitHub as Single Source of Truth (SSOT). Agents work independently but coordinate through GitHub Issues, following strict protocols to avoid conflicts and ensure efficient collaboration.
